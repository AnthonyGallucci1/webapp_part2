## 1. Project Files (Zip Archive)
- [ ] **Backend Code**: Ensure `server.js`, `models/`, `controllers/`, `routes/`, `config/`, `middleware/` are all included.
- [ ] **Configuration**: Include `package.json` and `.env`.
- [ ] **Verification Script**: Include `verify_api.js`.

## 2. External Design Document (EDD)
- [ ] **PDF Format**: Convert `EDD_Content.md` to PDF.
- [ ] **Wireframes**: Ensure the 4 wireframe images are included in the document.
- [ ] **Screenshots**: Add the 3 required screenshots (Server Start, API Verify, Project Structure).
- [ ] **Database Schema**: Ensure the schema table is present.

## 3. Product Backlog
- [ ] **Snapshot**: Take a screenshot of your Trello board (the one you just made).
- [ ] **PDF Format**: Save the screenshot as a PDF or include it in a document.

## 4. GitHub Repository
- [ ] **Push Code**: Ensure all code is pushed to your GitHub repo.
- [ ] **Link**: Copy the repository URL.

## 5. Demo Video
- [ ] **Slides**: Use the content from `Slide_Deck_Content.md`.
- [ ] **Recording**: Record a 5-10 min video showing:
    - Slide 1 (Team Info)
    - Slide 2 (Team Photos)
    - Demo: Run `npm run dev` and `node verify_api.js`.
    - Code Walkthrough: Briefly show the code.
- [ ] **Upload**: Upload to YouTube and get the link.

## Final Submission
- [ ] Zip all project files.
